# Task 4: Automate Deployment Using GitHub Actions + Ansible
- GitHub Actions will:
- Trigger Terraform to create EC2.
- Run an Ansible Playbook to:
- Install Node.js and npm
- Clone the React repo
- Install app dependencies
- Start the app
